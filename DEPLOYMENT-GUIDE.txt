## ğŸ†˜ Troubleshooting

### "Cannot read verses" error

**Cause:** Blob storage not enabled
**Fix:** Go to Netlify â†’ Storage â†’ Enable Blob Storage

### Verses not syncing

**Cause:** Network issue or function not deploying
**Fix:** 
1. Check Netlify deploy logs for errors
2. Make sure all 4 files were uploaded correctly
3. Check browser console for errors (F12)

### Function not found (404 error)

**Cause:** Functions folder not set up correctly
**Fix:**
1. Ensure `netlify/functions/verses.mjs` exists in GitHub
2. Ensure `netlify.toml` is at root level
3. Redeploy from Netlify dashboard

### Old verses still showing

**Cause:** Browser cache
**Fix:** 
1. Hard refresh: Ctrl+Shift+R (PC) or Cmd+Shift+R (Mac)
2. Clear browser cache
3. Switch tabs to force refresh
